# Sugoroku

CUIベースのすごろく

## 開発環境

CentOS 7.3 / 64-bit
gcc version 4.8.5 

## 実行方法

* make
* ./main

## 削除方法

ディレクトリを削除してください

## 遊び方

### START か QUIT か選ぶ場面

* "1[ENTER]" で開始します
* "0[ENTER]" で終了します

### すごろくする場面

開始時、ゴールまでのマス数を指定されます。
サイコロをふってゴールを目指してください。
ゴールするとめでたくゲームオーバーです。

* "1[ENTER]" でサイコロをふります
* "0[ENTER]" でやめます
